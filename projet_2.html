Portfolio XD

Ce projet consistait à créer un design de son propre portfolio avec Adobe XD. J'ai travailler dure sur le UI et le UX pour que l'Utilisateur sois très à l'aise à utiliser mon site Web. Le style est très épuré et simpliste. La navigation, la plupart du temps, se fait par glissement de souris. Par exemple, sur la page principale, il faut glisser vers le bas pour aller vers le bas. Dans la page À propos, il faut glisser une fois, mais ensuite cliquer sur voir plus pour continuer. Cliquer sur les images pour découvrir le prototype.
